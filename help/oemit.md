# @oemit

Emit a message to everyone in your current room except yourself. No actor attribution.

## Syntax

```
@oemit <message>
```

## Notes

- Lock: `connected` — no builder flag required.

## Examples

```
@oemit The door creaks open.
@oemit A cold wind sweeps through the room.
```
