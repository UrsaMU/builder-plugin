# @name

Rename an object. Checks for name collisions. Clears the moniker when a player is renamed.

## Syntax

```
@name <target>=<newname>
```

## Notes

- Maximum 200 characters.
- Renaming a player clears their display moniker.

## Examples

```
@name here=The Grand Library
@name #5=Rusty Sword
@name me=Alaric
```
