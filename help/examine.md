# @examine

Show detailed information about an object: flags, owner, lock, location, home, description, exits, contents, and attributes. Defaults to the current room.

## Syntax

```
@examine
@examine <target>
@ex <target>
```

## Examples

```
@examine
@examine me
@ex #12
@examine The Tavern
```
